# Student Management API

## How to run locally
1. npm install
2. npm start
3. Server runs on http://localhost:3000

## Deployed API URL
https://student-api-shahin.onrender.com

## Endpoints
### POST /api/students
Create a new student

### GET /api/students
Returns all students

## Tech used
Node.js, Express.js, UUID, Render Deployment
